# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-12

### Added
- Initial release
- Polar Radar visualization
- 3D LiDAR visualization with tunnel effect
- Robot FOV (First Person View) radar
- Object Detection radar with zone classification
- Control Center with switchable radar views
- Joystick widget for robot control
- Keyboard controls (WASD + arrows)
- Serial communication with Arduino
- Mini radar map widget
- Configurable settings
- Dark theme UI

### Features
- 4 different radar visualization styles
- Real-time data streaming from HC-SR04 sensor
- Radar mode (scan only) and Control mode (movement + scan)
- Speed control slider
- Connection status indicator
- Distance and angle display
